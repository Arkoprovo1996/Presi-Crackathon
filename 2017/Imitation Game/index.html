Imitation Game!



Task 1: May the Force be Full Brute

Your favourite teacher has sent a text, via a FILE to the most opinionated classmate of yours. The Curious Cat that you are, you wish to know it's contents. However, the file is encrypted.

It is known that your teacher always uses a Four Digit Number as zir keyphrase/password, using her Custom Asymmetric Cryptosystem, whose Source Code is Publicly Available.

It should be assumed, that the Original Message, i.e. the plaintext contained only the letters, A - Z, a - z & punctuation marks, ".", "," or "?" ( without quotes ).

The required Files are as follows:

  • Secret Message
  • Cryptosystem binary for Linux 64-bit
  • Cryptosystem Source Code


Task 2: Crash It Now!

The need of this hour is to fill your computer with practically infinite number of files of a small size, to make it impossible for others to find stuff in your computer or create their own files.

Write A Program ( WAP ) or a Scipt or a Combination of Several, to fill a folder with practically infinite files, with your own message. You may or may not create the delivery mechanism. ( e.g. autorun.inf files on Windows )
