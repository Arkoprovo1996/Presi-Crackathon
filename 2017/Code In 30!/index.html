Question 1: A Series of Unfortunate Numbers ...

Robert Langdon has again lost his memories. This time, instead of a fake hospital, he's been trapped inside a hidden chamber in Smolenskoye Cemetery. The key to his freedom, lies in three of numerous numbered rocks, which need pushed in the correct order. Unfortunately, he's forgotten the numbers. Fortunately, he has the following series written on the floor:

2, 3, 5, 6, 9, 11, 12, 16, 19, 21, 22, 27, 31, 34, 36, 37, 43, 48, 52, 55, 57, 58, 65, 71, 76, 80, 83, 85, 86, 94, 101, 107, 112, 116, 119, 121, 122, 131, 139, 146, 152, 157, 161, 164, 166, 167, 177, 186, 194

Unfortunately again, he's wrongly dialed your number. Help him out of the situation, by deciphering the given Integer series and find the next Three Terms of the series.


Question 2: Find the Function

Your University required a Simple Integer Hashing Program to create Hashes of the Students' Roll Numbers, to be used as WiFi Passwords. Before anyone could, Proprietary Inc. had provided them with it's own program. Luck has touched you today, as you found the Old Student Data file sent to your mail, by mistake.

It is a widely known fact that Proprietary Inc. uses three functions to HASH Data:

  • BITWISE Complementation ( ~ ): The bits of the function y = f(x) are complemented.
    • For instance, a signed 8-bit 110, i.e. ( 0000 0001 )2 will, on complementation yield, -110, i.e. ( 1111 1110 )2
  • INCREMENTATION By 16 ( + 16 ): The number is incremented by 1610
  • SQUARING ( x2 ): The number is squared.

However, it is unknown to you in which order these functions are called upon the given data, which is in the format [ ROLL NO.    HASH ], ( ROLL NO. TAB HASH ).

Find the Order in which these functions were called.

Example: If the functions are f(x), g(x) & h(x), report if it is f(g(h(x))) or g(f(h(x))) or so on ...


Question 3: Target Therapy

The veins & arteries of the body form a giant network, which can be used as a roadway to deliver drugs to treat diseases in specific regions of the body. The intersections of the veins, including organs can be thought of as nodes in a graph, the non-linear data-structure. The best path for the drug delivery, is obviously the shortest one between the injection point & the target point, be it a tear or a tumor.

You're given a set of 10 nodes, of which the first node is the injection point, the second, the target point & a set of connected nodes, with their inter-nodal distances, a.k.a path lengths, specified.

Find the shortest path between the Injection Point & the Target Point, in least time.

Example Input & Output:

Input:- 
		A F
		A B 2
		A C 1
		A H 4
		B H 1
		C D 1
		D F 3
		D G 5
		D H 3
		E F 8
		F H 1
		F I 3
		F J 2
		G J 1
		I J 3
Output:- 
		ABHF

In case of more than one shortest path, give any or both at your own discretion.
